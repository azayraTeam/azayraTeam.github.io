SimplyPrivate 3.6.11

Intro

SimplyPrivate 3.6.11 is a highly anticipated update, packed with improvements that will make your browsing experience faster and smoother. After some issues with version 3.6.1, we focused on fixing everything and bringing a more beautiful and functional interface. Let's see what changed ü§©

What Changed?

This version is all about performance and stability. A lot of things were updated, fixed, and even removed to improve the overall performance. Here are the most notable changes:

  • Updated JAF: Now SimplyPrivate runs faster with the latest version of JAF (JustAFrogScript). Everything will feel smoother!
  • New Window/Popup Effects: It's no longer just a plain window; now it has a cool background blur effect that makes everything look much nicer.
  • Fixed Visual Issues: Themes and colors were adjusted to make it easier on the eyes and to make the environment more comfortable.
  • Fixed "Unknown Errors": That annoying error that made the browser crash when something failed? That's gone now!
  • Removed Shortcuts: We know this was a handy feature, but we had to remove it for now due to compatibility issues. It will come back in a future update!
  • Re-added Intro Animation: Remember that intro from version 3.6.0? Well, it's back with a fresh blue accent to give everything a more refreshing feel.
  • Updated Plugin Explorer: The whole plugin explorer system got a redesign, looking cleaner and more organized. Some features might be unavailable depending on your operating system, but we're working on that.
  • Updated Settings Command System: Instead of an impractical window, we now have a dropdown menu with all available commands!
  • Added Window Flexibility: You now have the freedom to adjust your window width, and the design will adapt to fit your needs!
  • Improved Overall Stability: We removed some development tools and made other tweaks to ensure a faster and smoother experience.
  • Fixed Internet Manager Math: In older versions, the math for calculating G's based on internet speed was totally wrong. We've fixed it by switching from jaf(argm) to jaf(basic).
  • Updated Reset Speed: Now, "delete all data" is much faster without using the file queue.
  • Added New Theme "Quartz": A smooth pastel purple-themed design for both light and dark mode users.
  • Fixed Missing Tab Icons: We switched to an API used by Google, so rest assured, it doesn't track anything.
  • Removed Unused Assets: Goodbye to all the unused files that were just taking up space!
  • Updated "Enjoy" Screen: That old blinding screen is now replaced with a friendly, simple animation!

How Does It Compare to Previous Versions?

When we compare it to version 3.6.1, the difference is huge. In 3.6.1, the browser was full of bugs and slowdowns, but with 3.6.11, the performance is much more stable and faster. The visual improvements, like the background blur and the new color scheme, make everything look much more modern and comfortable.

However, some features, like the shortcuts, have been removed for now. But that's just to ensure that the current version is as stable as possible.
